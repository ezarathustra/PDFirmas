# PDFirmas

Aplicación desarrollada en **C#** para la gestión y firma de documentos PDF.

## Tecnologías

- C# (.NET)

## Funcionalidad

La funcionalidad del proyecto es agilizar el colocado de firmas para diferentes tipos de planos ya revisados previamente, este proyecto igual se podria adaptar para diferentes funcionalidades a la hecha.

## Estructura del proyecto

```
PDFirmas/
└── ProyectoPDF/
    └── *.cs   # Lógica de carga, edición y firma de documentos PDF
```

## Cómo ejecutarlo

```bash
git clone https://github.com/ezarathustra/PDFirmas.git
cd PDFirmas/ProyectoPDF
dotnet restore
dotnet run
```

## Motivación

El principal motivo por el cual cree el proyecto se debio a la oprtunidad de una necesidad, ya que hay un exceso de docuemtos ya revisados y la tardansa de firmarlos era enorme.
